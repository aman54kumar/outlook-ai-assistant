An AI assistant that lives in your inbox — and keeps your email on your machine.

Summarize a thread, draft a reply, or fix the tone of a message without leaving Outlook. Runs on a local model by default, so sensitive email never leaves the device.

How it works

Private by default

In the real add-in, requests go to a local Ollama model through a server on your own machine. The cloud key, if you use one, never reaches the browser.

Bring your own model

One toggle switches between local and a cloud provider. The provider layer supports Ollama, OpenAI-compatible endpoints, and Anthropic.

Native to Outlook

Built with Office.js. It reads the open email, and inserts results straight into your draft while composing.

This page is a front-end demo. The full local-first version runs inside Outlook — see the README to run it.
